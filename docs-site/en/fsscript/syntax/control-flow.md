# Control Flow

## if Statement

```javascript
if (condition) {
    // code
} else if (condition2) {
    // code
} else {
    // code
}
```

## switch Statement

```javascript
switch (expression) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // default code
}
```

## for Loop

```javascript
// C-style for
for (var i = 0; i < 10; i++) {
    // code
}

// for-in (iterate object keys)
for (let key in obj) {
    // code
}

// for-of (iterate array values)
for (const value of array) {
    // code
}
```

## while Loop

```javascript
while (condition) {
    // code
}
```

## Loop Control

```javascript
break;          // Exit loop
continue;       // Skip current iteration
```

## Ternary Operator

```javascript
condition ? valueIfTrue : valueIfFalse
```

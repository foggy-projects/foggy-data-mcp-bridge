# Introduction

Foggy Dataset is an embedded semantic layer framework for declarative dimension modeling.

## Features

- **JM (Join Model)**: Define dimension tables, fact tables, and their relationships
- **QM (Query Model)**: Define reusable query templates with dynamic parameters
- **Multi-Database Support**: MySQL, PostgreSQL, SQL Server, SQLite
- **Automatic SQL Generation**: Generate optimized SQL from high-level queries

# Introduction

Foggy MCP Service provides Model Context Protocol (MCP) integration for AI assistants.

## Features

- **MCP Protocol**: Standard JSON-RPC 2.0 interface for AI tools
- **Multiple Endpoints**: Analyst, Admin, and Business user roles
- **Natural Language Query**: AI-powered data analysis
- **Claude & Cursor Integration**: Works with popular AI assistants

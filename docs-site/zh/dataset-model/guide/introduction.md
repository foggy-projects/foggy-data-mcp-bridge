# 简介

Foggy Dataset 是一个嵌入式语义层框架，用于声明式维度建模。

## 特性

- **JM (Table Model)**: 定义维度表、事实表及其关联关系
- **QM (Query Model)**: 定义可复用的查询模板，支持动态参数
- **多数据库支持**: MySQL、PostgreSQL、SQL Server、SQLite
- **自动 SQL 生成**: 从高级查询自动生成优化的 SQL

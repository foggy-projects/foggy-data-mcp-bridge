# description_model_internal 工具描述

## 基本信息
- **工具名称**: description_model_internal  
- **功能**: 获取指定模型的详细元数据，包括所有字段定义和字典信息
- **调用方式**: 内部工具 (T2/M2 使用)

## 工具描述

获取指定数据模型的完整元数据定义，包含所有字段详情、数据类型、字典映射等信息。

📋 **详细功能**:
- 返回模型的完整字段列表及其属性
- 提供每个字段的详细描述和数据类型
- 包含字典字段的完整值映射关系
- 展示字段的业务含义和使用规则
- 说明维度字段的$id和$caption用法

🔍 **使用场景**:
- 用户需要了解某个模型的所有可用字段
- 查询前需要确认字段名和数据类型
- 需要查看字典字段的所有可选值
- 了解特定模型的数据结构
- 确认字段的查询和过滤规则

📊 **返回内容结构**:
```json
{
  "model": "模型名称",
  "description": "模型描述",
  "fields": [
    {
      "name": "字段名",
      "type": "数据类型",
      "description": "字段描述",
      "dictionary": "字典名称(如果是字典字段)",
      "values": {...} // 字典值映射
    }
  ],
  "dictionaries": {...} // 所有字典定义
}
```

💡 **特殊说明**:
- 维度字段支持$id（返回ID值）和$caption（返回显示名称）后缀
- 字典字段可以不加后缀（返回原始值）、加$id（返回字典值）或$caption（返回字典名称）
- 度量字段用于聚合计算（SUM、AVG、MAX、MIN、COUNT等）

⚙️ **内部实现**: 调用 GET /metadata/model/{model} 接口获取模型详细元数据

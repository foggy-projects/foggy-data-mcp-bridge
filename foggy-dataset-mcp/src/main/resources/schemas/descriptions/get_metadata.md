# get_metadata 工具描述

## 基本信息
- **工具名称**: get_metadata
- **功能**: 获取用户级元数据包，包含可访问的模型列表和常用字段信息
- **调用方式**: 内部工具 (T2/M2 使用)

## 工具描述

获取当前用户可访问的所有数据模型和字段的元数据信息包。

📊 **主要功能**:
- 返回用户有权限访问的所有数据模型列表
- 提供每个模型的基本信息和字段说明
- 包含字段的数据类型、字典映射等元数据
- 展示字段在不同模型中的语义关联

🎯 **使用场景**:
- 用户询问"有哪些可用的数据模型"
- 用户想了解某个字段的含义和用法
- 需要查看字典字段的可选值
- 探索不同模型间的字段关联

💡 **返回内容**:
- 模型列表及其描述
- 字段的语义索引
- 字典附录（字段值映射）
- 字段类型说明（维度、度量、属性等）

⚙️ **内部实现**: 调用 GET /metadata/user 接口获取用户级元数据包
